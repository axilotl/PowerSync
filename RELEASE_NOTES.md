## What's Changed

**Align electricity price chart to the current time**
The built-in PowerSync dashboard now extends each electricity price history series to the current render time, so the import and export price dots sit on the same time position as the blue current-time line. This fixes stale-looking export/import positions when Home Assistant history only contains the last price-change timestamp.

Update available via HACS
