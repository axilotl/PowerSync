## What's Changed

**Toggle dashboard chart series**
The Home Assistant dashboard chart legend is now interactive, so series such as Solar, Grid, Battery, Home, Import Price, and Export Price can be toggled on or off directly from the graph. Hidden series are removed from the active plot scale and tooltip, making noisy lines easier to isolate.

**Midnight-to-midnight 24-hour charts**
The Energy - 24 Hours and Electricity Prices - 24 Hours cards now use the local day window from midnight to midnight instead of a rolling last-24-hours range, so today’s usage and prices line up with the intended daily view.

Update available via HACS
