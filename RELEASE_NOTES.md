## What's Changed

**Dashboard flow and layout fixes**
The Home Assistant dashboard power-flow animation now keeps existing flow-line animation state when HA sends routine state updates, avoiding the visible restart/jitter effect. Customize layout mode also now saves the actual lane and drop position for each card, so dragged dashboard sections stay where they are dropped instead of being rebalanced away.
