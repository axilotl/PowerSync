## What's Changed

**Show Maximum grid import in battery settings**
The Smart Optimization Maximum grid import setting now appears on the combined battery settings screens for Tesla, Sigenergy, Sungrow, FoxESS, and GoodWe. This fixes the case where the backend supported the new site import cap but the option was not visible when reconfiguring battery optimization settings.

Update available via HACS
