## What's Changed

**24-hour action plan now scrolls through every action**
The dashboard optimisation card no longer hides action-plan entries behind a non-clickable "+ more actions" placeholder. The 24-hour action list now renders the full plan in a bounded scroll area, so longer schedules remain viewable without pushing the rest of the dashboard out of reach.
