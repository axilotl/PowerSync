## What's Changed

**Globird tariff setup guidance for non-Tesla systems**
PowerSync now clearly scopes the Tesla app tariff prerequisite to Tesla Powerwall users only. Sigenergy, FoxESS cloud, and other non-Tesla systems are directed to configure Globird/TOU custom tariff rates inside PowerSync, so setup guidance matches the pricing path used for price sensors, battery optimisation, and EV charging.

**Dashboard layout customization drag fixes**
The custom dashboard layout editor now uses a dedicated drag surface and keeps the customize toggle text in sync, improving drag-and-drop behaviour when rearranging cards on desktop and touch devices.
