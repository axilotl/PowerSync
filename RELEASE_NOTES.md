## What's Changed

**Flow Power: Tariff Selection Replaced with Single Combined Dropdown**
The network and tariff code were previously two separate dropdowns, requiring users to first pick a DNSP and then wait for a re-render to see tariff codes — which proved unreliable and often showed no codes at all. Both steps are now merged into a single dropdown that loads all region-appropriate network and tariff combinations upfront (e.g. "Ausgrid — RESELE"), covering both the initial setup wizard and the options flow network screen.
