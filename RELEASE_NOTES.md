## What's Changed

**Two-press confirmation on Pair Powerwall Gateway when already paired**
First-time pairing stays single-press — no friction in the common setup flow. But if the site is already paired, pressing **Pair Powerwall Gateway** now posts a "press again within 30s to confirm" notification instead of immediately starting a re-pair. A second press in the window kicks off the new key registration. Stops a stray automation or accidental dashboard tap from spamming Tesla with new keys and prompting an unwanted DC-isolator toggle. The Unpair button's two-press safety from 2.12.225 is unchanged.
