## What's Changed

**Amber Site Selection — Always Visible**
The Amber site dropdown now appears for all users in both the initial setup and the options flow (Configure → Pricing → Amber), not just users with multiple sites. Previously the dropdown was gated on having more than one site, so single-site users had no way to confirm or change which NMI was configured. The site picker is now always shown when the Amber API returns at least one site.
