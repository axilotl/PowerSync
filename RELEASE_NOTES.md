## What's Changed

**Dashboard chart scale and layout fix**
The Home Assistant dashboard now scales price chart history values before generating the axis, so import/export prices display correctly in c/kWh instead of collapsing to repeated 0 c/kWh labels. All native dashboard graphs now include hover/touch tooltips, and the lower forecast charts move into the shorter controls column on wide dashboard layouts to reduce the empty space visible on landscape tablets and desktop views.

Update available via HACS
