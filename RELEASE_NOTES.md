## What's Changed

**Dashboard chart tooltips**
The Smart Optimization dashboard now shows hover and touch tooltips on the SOC and Battery Power graph, including SOC, charge, home load, export, EV charging, calculated reserve, and IDLE hold values at the selected time. The electricity price graph in the same optimizer card also now shows matching import and export price tooltips.

**Consistent optimizer graph interaction**
The optimizer charts now use the same vertical guide and tooltip behavior as the other native PowerSync chart cards, making it easier to inspect planned battery actions and prices directly from the Home Assistant dashboard.
