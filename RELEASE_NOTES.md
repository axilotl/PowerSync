## What's Changed

**Flow Power current price display for Sigenergy systems**
PowerSync now stores the canonical Flow Power tariff schedule during Sigenergy tariff syncs, so the standard current import/export price sensors use the same PEA-adjusted 30-minute tariff that is uploaded to Sigenergy instead of falling back to the raw KWatch/AEMO wholesale price.
