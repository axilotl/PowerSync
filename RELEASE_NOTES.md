## What's Changed

**Align the TOU schedule current-time marker**
The TOU Schedule dashboard marker now uses the real current timestamp while keeping the active slot price, so the dot and vertical now-line align with the actual time instead of the previous tariff boundary.

**Clean up the TOU chart axis**
The TOU chart no longer renders the zero-price tick label where it can read like a stray x-axis label during free import periods.
