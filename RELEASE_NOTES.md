## What's Changed

**Profit Max setting changes now re-optimise immediately**
Changing Profit Max mode, target time, or target SOC now refreshes the Smart Optimization plan as soon as the setting is saved when the optimiser is enabled. This keeps the active schedule aligned with the new Profit Max behaviour instead of waiting for the next scheduled optimisation cycle.

Update available via HACS
